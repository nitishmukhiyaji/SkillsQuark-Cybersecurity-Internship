## 📌 About This Repository

This repository showcases the practical tasks completed during the SkillsQuark Cybersecurity Internship Program.

The work focuses on the complete penetration testing lifecycle, starting from reconnaissance and network discovery, moving through packet-level traffic analysis, and finally validating vulnerabilities through controlled exploitation techniques.

Each assessment was conducted in an isolated virtual laboratory using industry-standard cybersecurity tools and documented through professional technical reports.

# 🔍 Task 1 — Reconnaissance & Nmap Assessment

### Mission

Identify live hosts, discover exposed services, enumerate versions, and analyze the attack surface of the target environment.

✔ Host Discovery

✔ TCP Port Scanning

✔ Service Enumeration

✔ Version Detection

✔ Operating System Identification

✔ Vulnerability Mapping

* Security Documentation

# 🌐 Task 3 — Network Sniffing & Traffic Analysis
